C++ & Qt Projects

Color Mixing Game

Collaborated in a team to develop a color mixing game with three levels. Each level has three stages, with four swatches to match. There is a bit of an error tolerance based on the euclidean distance in color space, as well as an API call to name each color in the cauldron. Box2D was used to add interactivity to the game.

Sprite Editor

Created a sprite editor using Qt. This invovled allowing a user to select different brush colors, brush sizes, canvas sizes, as well as an animation preview with the ability to change the animation speed. The user is able to click on different frames, make new frames, and use the onion skin feature to reference other frames. The user can also save as a JSON file and load previous sprite projects in.

Simon Game

Recreated the Simon memory game using Qt. Involved timer structure and signal/slot architecture as well as game logic.

Wavy Text Generator

Simple wavy text generation using Haru library.
